**Convert Any File, Effortlessly.**
A premium, open-source web application for converting files between formats instantly. Built with Flask and a stunning glassmorphic UI.

![FormatWave Hero](https://github.com/hackwithvishnu/FormatWave/blob/main/static/hero.png?raw=true)

## ‚ú® Features

- **Premium UI:** Dark mode, glassmorphism, and smooth animations.
- **Drag & Drop:** Easy file upload with visual feedback.
- **7+ Formats:** Supports PDF‚ÜíPNG, WebP‚ÜíPNG, PNG‚ÜíWebP, JPG‚ÜíPNG, and more.
- **Batch Processing:** Convert multiple files at once.
- **ZIP Download:** Download all converted files in a single ZIP archive.
- **Privacy Focused:** Files are auto-deleted after 1 hour.
- **Responsive:** Works perfectly on mobile, tablet, and desktop.

## üöÄ Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** Vanilla HTML, CSS (Custom Design System), JavaScript
- **Processing:** PyMuPDF (PDFs), Pillow (Images)

## üõ†Ô∏è Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/hackwithvishnu/FormatWave.git
    cd FormatWave
    ```

2.  **Create a virtual environment (optional but recommended)**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```

3.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the application**
    ```bash
    python3 app.py
    ```

5.  **Open in browser**
    Visit `http://localhost:5000` to start converting files!
